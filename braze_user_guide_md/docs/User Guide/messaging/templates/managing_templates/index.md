# Manage templates

> Archiving or duplicating templates can help better organize and manage them. This reference article covers how to archive and duplicate templates in the **Templates** section of the Braze dashboard.

## Duplicating templates




![Dropdown menu with duplicate option.](https://www.braze.com/docs/assets/img/template_duplicate_cog.png?d6054bbc3072188e5319599a5bba449b){: style="float:right;max-width:15%;margin-left:15px;"}

To duplicate an individual template, select <i class="fas fa-ellipsis-v"></i> **More options** for the template, then select **Duplicate** from the dropdown menu.
<br><br>

**Note:**


For [Content Block](https://www.braze.com/docs/user_guide/messaging/design_and_edit/content_blocks/) templates, a draft copy is created. For all other templates a new duplicate copy is automatically created.








Duplicating multiple templates can be achieved by selecting the checkbox next to the template name. First, select the templates and then select **Duplicate**.

Duplicated templates can be found by sorting the **Last Edited** column. By default, new templates will be named `Copy of ORIGINAL_TEMPLATE_NAME`.



![Three templates sorted by the time the templates were last edited, with a copied template at the top of the list.](https://www.braze.com/docs/assets/img/duplicate_multiple_template.gif?5090371ef2f93a5085bd93ed034ccd9e)




## Archiving templates

![Expanded settings dropdown menu that shows three options: "Archive", "Duplicate", and "Copy to workspace" with the "Archive" option highlighted.](https://www.braze.com/docs/assets/img/template_archive_cog.png?51b0a55f5dba6c7dd81b27e2590ec453){: style="float:right;max-width:20%;margin-left:15px;"}

To archive an individual template, select <i class="fas fa-ellipsis-v"></i> **More options** on the template grid screen and select **Archive**. When a template is archived, note the following different scenarios:

- Active campaigns continue to use the archived template without any interruption.
- Draft campaigns retain the archived template's content and can be edited and launched.
- To edit an archived template, you must unarchive it first. Similarly, to use an archived template for a campaign, you must unarchive the template first.

To archive multiple templates, select the checkbox next to each template that you want to archive. After you've selected multiple templates, select **Archive**. You can find your archived templates by selecting **Archived** under **Show** in the template grid.

![Saved Drop & Drop Email Templates section that shows two selected templates and toolbar with the option to archive.](https://www.braze.com/docs/assets/img/archive_multiple_template.png?67bfa25bf9953d67e8a1fb5109e645ac){: style="max-width:60%;"}

**Important:**


Archiving is not currently available for [link templates](https://www.braze.com/docs/user_guide/messaging/templates/email_templates/link_aliasing/#link-templates).
